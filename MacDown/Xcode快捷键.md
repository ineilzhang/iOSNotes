
### Xcode快捷键

@(学习笔记本)

#### 1.工程导航器 command + 1
#### 2.显示和隐藏导航器面板 command + 0
#### 3.显示和隐藏实用工具面板 command + option + 0
#### 4.在辅助编辑器中打开文件 option + left-click + file
#### 5.搜索导航器 command + shift + F
#### 6.文件跳转栏 control + 6
#### 7.快速打开 command + shift + O
#### 8.在.h和.m文件之间快速切换 control + command + 上下方向键
#### 9.运行APP command + r
#### 10.清除工程 command + shift + K
#### 11.构建应用程序 command + b
#### 12.文档和参考 command + shift + zero
#### 13.特殊字符 control + command + 空格
